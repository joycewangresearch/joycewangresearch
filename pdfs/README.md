# PDFs Folder

Place all 47+ research publication PDF files here.

## How to Get the PDFs

1. Go to https://www.joycewangresearch.com/publications-1
2. Download each PDF by clicking on the thumbnails
3. Save them with descriptive names that match the links in `publications.html`
4. Upload all PDFs to this folder in your GitHub repository

## Suggested File Naming

Use lowercase with hyphens, matching what's in the HTML:
- `wang-et-al-2013-quantum-cognition.pdf`
- `wang-busemeyer-2013-qq-model.pdf`
- etc.

Or keep the original file names from the website.
